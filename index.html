如何使用grunt-wiredep.

这是啥

Grunt 棒极了.

Bower 棒极了.

现在他们可以很好地协同工作。

grunt-wiredep 是一个grunt插件，它可以找到你的组件然后插入到指定的html文件中去。

就像下面步骤所展示的，不论你是已经开始使用Bower和Grunt还是刚刚接触，grunt-wiredep都非常容易集成到项目中。

请注意: Bower还很年轻，所以变化非常快。要想让grunt-wiredep正常工作，bower组件的作者必须遵循特定的约定和最佳实践。不用说，一些bower组件可能不能被正常的注入。

首先

要使用Bower相关的依赖, 在你的html文件中插入一个注释块： 

  <!-- bower:js -->
  <!-- endbower -->

接下来

剩下的就交给Grunt 和 grunt-wiredep 。先安装它们： 

  npm install grunt-wiredep --save-dev

现在

打开你的Gruntfile，加入下面的代码： 

  grunt.loadNpmTasks('grunt-wiredep');

  grunt.initConfig({
    wiredep: {
      target: {
        src: 'index.html' // point to your HTML file.
      }
    }
  });

接下来

假设你的应用需要用到jQuery

像往常一样，通过bower安装jQuery 

  bower install jquery --save

然后打开命令行，输入： 

  grunt wiredep

最后

查看你的html文件，应该类似于下面： 

  <!-- bower:js -->
  <script src="bower_components/jquery/jquery.js"></script>
  <!-- endbower -->
